# @tractorbeam/typescript-config

Shared `tsconfig` presets for Tractorbeam projects.

## Install

```bash
pnpm add -D @tractorbeam/typescript-config
```

## Usage

In your projectâ€™s `tsconfig.json`:

```json
{
  "extends": "@tractorbeam/typescript-config/node22.json"
}
```
